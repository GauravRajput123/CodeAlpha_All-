# 🎮 Hangman Game - CodeAlpha Internship

## 📌 Project Description
This is a simple text-based Hangman game built using Python. 
The player guesses a randomly selected word letter by letter with a maximum of 6 incorrect attempts.

## 🚀 Features
- Random word selection
- 6 incorrect guess limit
- Console-based interaction
- Beginner-friendly logic

## 🛠 Technologies Used
- Python
- Random module
- Loops & Conditional Statements

## 🎯 Concepts Covered
- Strings
- Lists
- While Loop
- If-Else
- User Input Handling

## 📂 How to Run
```bash
python hangman.py
